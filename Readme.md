浪游蜘蛛纸牌
        游戏开始时，发有十叠牌，每叠中只有一张正面朝上。其余的牌放在窗口右下角的五叠牌叠中；新一轮发牌时用这些牌。
        移牌的方法是将牌从一个牌叠拖到另一个牌叠。移牌的规则如下：
        可以将牌叠最底下的牌移到空牌叠。
        可以将牌从牌叠最底下移到牌值仅次于它的牌上，不论牌套或颜色如何。
        可以像对待一张牌一样移动一组同样牌套、依序排好的牌。
        准备新一轮发牌时，请单击“发牌”，或者单击窗口右下角的牌叠。
        在新一轮发牌之前，每一叠中都必须有牌。
